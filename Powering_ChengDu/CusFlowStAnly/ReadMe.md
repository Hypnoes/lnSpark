# 用电客户流入流出情况分析

对该模块的分析，_房屋空置率是研究的重点_，可以从单个家庭用户的电能数据入手来作为判断房屋空置的依据. 
<br>
<br>
__拟采用横纵对比的方法:__
* 纵向对同一区域 ``以住宅小区为例`` 内用户用电数据进行长时间的差别比较，作为纵向用户流入流出的判断方法
* 横向与对不同区域 ``比如入住率比较高的小区`` 典型用户用电情况做比较，最终得到判断结果
